### Instalação
- Node.JS
- NPM/Yarn

1. Instalar typescript globalmente: `sudo npm i -g typescript`
2. Versão do typescript: `tsc -v`
3. Instalar typescript no node: `sudo npm i -g ts-node`
4. watch dos arquivos ts `tsc -w`
